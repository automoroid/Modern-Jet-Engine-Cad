# Modern High-Bypass Turbofan Engine

This repository contains a detailed 3D CAD model of a modern jet engine. The design focuses on the architecture of high-bypass turbofans used in commercial aviation, emphasizing the mechanical relationship between the cold fan section and the hot power core.

## 🛠 Project Overview
The model follows the **Brayton Cycle** ("Suck, Squeeze, Burn, Blow") and includes the following key sections:

*   **N1 Stage:** Large-diameter front fan and low-pressure turbine assembly.
*   **Compressor Core:** High-pressure stages designed for optimal air compression.
*   **Combustion Chamber:** Internal geometry for simulated fuel-air mixing.
*   **Turbine Section:** High-temperature blade profiles that drive the propulsion system.

## 📊 Technical Specifications
*   **Engine Type:** High-Bypass Turbofan
*   **Components:** Full assembly including housing, rotors, and stator vanes.
*   **Primary Goal:** Educational visualization and mechanical design study.

## 📂 File Formats
*   `.STEP` / `.STP`: For universal CAD compatibility (SolidWorks, CATIA).
*   `.STL`: Optimized for 3D printing and mesh viewing.
*   `.PNG`: High-resolution renders of the final assembly.

## 🚀 How to Use
1. **Clone** this repository to your local machine.
2. Open the `.STEP` file in your preferred CAD software to view the assembly tree.
3. Refer to the `/renders` folder for cross-section views of the internal components.
